# JetpackDataStore
Jetpack DataStore Example

DataStore 是google新出的用于代替SharePreferences的数据存储，有两种实现方式，一种是Preferences DataStore，一个是Proto DataStore
Preferences DataStore的详解地址：https://juejin.cn/post/6909441104194748423
Proto DataStore的详解地址：https://juejin.cn/post/6910423905786298382

该项目是DataStore两个实现方式的简单事例
